ZenKey Sign In Buttons

You can use these ZenKey sign in buttons to start the sign in process. Just copy the HTML and CSS into your app.

The color version of the button should be used on white or light backgrounds. The light button variation should be used on dark backgrounds and in dark mode. You may also use the "Continue" call to action if it is more suitable to your use case. Micro buttons should be used when space is limited. Because the button does not contain a call to action, the user interface must make the button's function clear.
